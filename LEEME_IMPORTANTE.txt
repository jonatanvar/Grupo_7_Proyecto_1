DESCRIPCIÓN TÉCNICA:
Sistema de información completo desarrollado en Java (JDK 21/24) bajo el paradigma
de Programación Orientada a Objetos. Implementa patrón arquitectónico MVC,
persistencia de datos con SQLite y seguridad basada en roles (RBAC).

REQUISITOS DEL SISTEMA:
1. Java Development Kit (JDK) 17 o superior.
2. IntelliJ IDEA (Entorno recomendado).

------------------------------------------------------------------------
INSTRUCCIONES DE INSTALACIÓN (PASO A PASO)
------------------------------------------------------------------------

PASO 1: ABRIR EL PROYECTO
   - Descomprima la carpeta "Grupo_7_Proyecto_1".
   - Abra IntelliJ IDEA -> Open -> Seleccione la carpeta descomprimida.

PASO 2: CONFIGURACIÓN DE LIBRERÍAS
   Este proyecto incluye las librerías necesarias en la carpeta "/lib".
   Si al abrir el proyecto ve errores en rojo en los "import", haga esto:

   1. Vaya al menú: File > Project Structure > Libraries.
   2. Haga clic en el botón (+) > Java.
   3. Navegue dentro de la carpeta del proyecto a la carpeta "lib".
   4. Seleccione los dos archivos .jar que encontrará ahí:
      - flatlaf-3.4.jar (Para el diseño visual moderno)
      - sqlite-jdbc-3.45.1.0.jar (Para la base de datos)
   5. Haga clic en OK y luego en Apply.

PASO 3: EJECUCIÓN
   1. Navegue a: src/Proyecto_iglesia/app/Main.java
   2. Haga clic derecho en el archivo Main -> Run 'Main.main()'.
